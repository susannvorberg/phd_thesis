F.12 Statistics for Comparing Couplings computed with Pseudo-likelihood and Contrastive Divergence

Proteins with the 20th largest Neff values from the dataset of 300 proteins used to evaluate CD. Statistics have been computed for the contact scores computed from pseudo-likelihood couplings and CD couplings. Contact scores are computed as the APC corrected Frobenius norm of the couplings \(\wij\). CD coupling parameters have been optimized with SGD using the following hyperparameter settings: simgoidal learning rate schedule with initial learning rate \(\alpha_0 = \frac{5e-2}{\sqrt{N_{\text{eff}}}}\) and decay rate \(\gamma = 5e-6\). Regularization coefficient \(\lambda_w = 0.1L\). In order to approximate the gradient with CD, 0.3Neff sequences are randomly selected and evolved with a Gibbs chain for 1 step.
